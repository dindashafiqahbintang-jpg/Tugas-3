# Panduan Pengisian Formulir Pendaftaran Komunitas Teknologi

Formulir ini digunakan untuk mendaftarkan anggota baru pada **Komunitas Teknologi**. Berikut panduan singkat untuk mengisi form dengan benar:

## 1. Data Pribadi

* **Nama Lengkap**: Masukkan nama depan dan belakang secara lengkap.
* **Alamat Email**: Gunakan alamat email aktif yang dapat dihubungi.
* **Kata Sandi**: Minimal 8 karakter, gunakan kombinasi huruf dan angka disarankan.
* **Usia**: Masukkan angka sesuai usia Anda (15-100 tahun).
* **Tanggal Lahir**: Pilih tanggal lahir sesuai KTP atau identitas resmi.
* **Jenis Kelamin**: Pilih salah satu opsi (Laki-laki atau Perempuan).

## 2. Informasi Tambahan

* **Minat**: Centang bidang yang diminati (AI, Web, IoT, Game).
* **Nomor Telepon**: Masukkan nomor aktif (10-13 digit).
* **Link Portofolio**: (Opsional) Masukkan link website atau portofolio online.
* **Upload Foto Profil**: Unggah foto berformat gambar untuk keperluan identitas anggota.

## 3. Tombol Aksi

* **Reset**: Membersihkan seluruh data yang telah diisi.
* **Kirim**: Mengirimkan data formulir ke server atau halaman submit (sesuai pengaturan backend).

> Pastikan semua data yang diisi benar dan valid sebelum menekan tombol **Kirim**. Data yang lengkap akan memudahkan proses verifikasi dan registrasi anggota baru.

## 4. Catatan

* Formulir ini menggunakan HTML5 standar dan dapat dijalankan di browser modern.
* Pastikan file foto yang diunggah tidak lebih dari ukuran maksimal yang diperbolehkan (disesuaikan dengan server jika ada).
